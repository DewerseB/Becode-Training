Qu'est-ce que Markdown?

  • Markdown est un langage de balisage léger avec une syntaxe de formatage de texte brut.
  • Cela signifie qu'en utilisant seulement quelques symboles supplémentaires dans votre texte, Markdown vous aide à créer un document avec une structure explicite.
  • Par exemple, en texte brut, rien n'indique quel texte est en **gras**, en *italic* ou est un lien. Mais avec MarkDown, en utilisant un symbole de certaines manières, vous pouvez.

Ici une liste de Syntax ce que MarkDown peut faire:
